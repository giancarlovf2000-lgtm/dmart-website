# Media del sitio (fotos y videos)

Sube aquí los archivos reales y actívalos pasando la ruta al componente `<MediaSlot>`.
Mientras no existan, el sitio muestra un placeholder de marca elegante (no se ve roto).

## Nombres sugeridos
- `hero.mp4` (o `hero.jpg`) — fondo del Hero (recomendado 1920×1080, <8 MB para video).
- `hero-poster.jpg` — imagen de respaldo/poster del video del Hero.
- `categoria-belleza.jpg`, `categoria-salud.jpg`, `categoria-comercial.jpg`, `categoria-tecnico.jpg`
- `programa-<slug>.jpg` — p. ej. `programa-cosmetologia.jpg`
- `cta.jpg` — fondo de la banda CTA.
- `campus-barranquitas.jpg`, `campus-vega-alta.jpg`

Formatos: JPG/WebP para fotos, MP4 (H.264) para video. Optimiza el peso antes de subir.
